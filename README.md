# ZortilWatch

Незалежний самостійно-хостований застосунок віддаленого доступу та керування
комп'ютерами (аналог TeamViewer / AnyDesk / RustDesk), що повністю належить
власнику й не залежить від сторонніх хмар. Повна специфікація продукту —
у [PRD.md](PRD.md), дорожня карта робіт — у [TASKS.md](TASKS.md).

> **Статус:** MVP майже зібраний. Воркспейс із чотирьох Rust-крейтів
> (`core`, `desktop/src-tauri`, `web-wasm`, `service`) збирається; інтерфейс
> `ui/` та бекенд `server/` на місці. Налаштування середовища —
> у [docs/dev-setup.md](docs/dev-setup.md).

## Структура репозиторію

| Каталог | Роль |
|---------|------|
| `core/` | Продуктивне ядро на Rust: захоплення екрана, апаратне кодування/декодування відео, передавання вводу, мережа (WebRTC/обхід NAT), наскрізне шифрування. Спільне для всіх платформ. |
| `ui/` | Спільний інтерфейс (TypeScript + React). Використовується і в десктопі (через Tauri), і як веб-клієнт у браузері. |
| `desktop/` | Оболонка Tauri для Windows: загортає `ui/` і викликає `core/`. |
| `server/` | Самостійно-хостований бекенд: сервер з'єднань (rendezvous), ретранслятор (coturn/TURN), сервіс акаунтів та адресної книги. |
| `docs/` | Архітектура, зафіксовані рішення та правила (зокрема ліцензійні). |

## Архітектурні рішення

Шість несучих рішень фундаменту зафіксовано в
[docs/architecture.md](docs/architecture.md):

- **A1** — одне Rust-ядро для десктопа; веб-клієнт ділить інтерфейс і протокол, а медіаканал у браузері — на рідному WebRTC.
- **B1** — пароль підключення слугує криптодоказом (захист від MITM навіть проти власника сервера).
- **C1** — H.264 як універсальна підлога; VP9/AV1 як автоматичний бонус.
- **D1** — контроль якості за ціллю (різкість ↔ плавність); ручні повзунки як безпечні стелі.
- **E1** — «спершу напряму», ретранслятор як справжній запас.
- **F1** — ліцензійний фільтр за політикою (пермісивна родина + MPL; твердий блок GPL/AGPL/LGPL).

## Передумови розробки

| Інструмент | Призначення | Стан на цій машині |
|------------|-------------|--------------------|
| Node.js + npm | збірка `ui/` | ✅ Node 24, npm 11 |
| Git | контроль версій | ✅ 2.53 |
| Rust toolchain (rustup, MSVC target) | `core/`, `desktop/` | ✅ зафіксовано в `rust-toolchain.toml` |
| MSVC C++ Build Tools + Windows SDK | обов'язкові для Tauri / `windows-rs` | ⚠️ потрібні |
| WebView2 Runtime | рушій вебвʼю для Tauri | зазвичай уже в сучасній Windows |

Точні версії стека зафіксовано в `rust-toolchain.toml` і `package.json`.
Повний покроковий гайд із налаштування — у [docs/dev-setup.md](docs/dev-setup.md).
