# desktop/ — оболонка Tauri v2 (Windows)

Загортає спільний `ui/` у нативний застосунок і викликає `core/`. Tauri/wry/tao —
MIT/Apache. Деталі — [../docs/architecture.md](../docs/architecture.md).

- **WebView2** — рушій вебвʼю; уже присутній у Windows 10 1803+/11.
- **Бандли:** NSIS-інсталятор (типово) + офлайн-варіант інсталятора; MSI опційно (лише на Windows).
- ⚠️ **Портативний єдиний `.exe` із вшитим WebView2 неможливий.** «Портативність» реалізуємо як standalone-збірку, що покладається на наявний у системі WebView2 — рішення формалізуємо на Етапі 11.
- Для розповсюдження знадобиться **підпис коду** (OV/EV), інакше SmartScreen/антивіруси блокуватимуть.

**Статус:** очікує Rust toolchain + MSVC Build Tools (обов'язкові для Tauri на Windows).
